# hrpos-lib

### Purpose
House shared library functionality such as db models, api libraries

```
hrpos-lib
    └── paytronix
    └── engage
    └── monetary
    └── HeartlandAPI
    └── db-models
    └── models
```

| `central-api`  | :white_check_mark:  |
| `mb-ops`  | :white_check_mark:  |
| `lambda`  | :ballot_box_with_check:  |

```
import { HeartlandAPIFactory } from 'hrpos-lib/HeartlandAPI';
```
